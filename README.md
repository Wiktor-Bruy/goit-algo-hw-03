# Homework 3
This is one of the assignments I completed while learning Python. It contains three functions.
The first, get_days_from_today, accepts a date as a string and returns the number of days from that date to today. (If the date is in the future, the number of days is negative.)
The second, get_numbers_ticket, accepts three integer parameters: the minimum value, the maximum value, and the number. Based on the input, it generates an array of numbers in the specified range and number.
The third, normalize_phone, accepts a phone number in various formats as a string and normalizes it to the standard format Ж +38ХХХХХХХХХХ.
